Exercice des quatre cibles

Le point de départ est noté A et a pour coordonnées (10, 20).

Les quatre cibles sont nommées B, C, D et E dans cet ordre et en partant du haut. Elles ont pour coordonnées :

  • B(250, 390)
  • C(250, 200)
  • D(250, 20)
  • E(250, -100)

# Espace de jeu
taille = 300
x_pos_axe, y_pos_axe = 200, 200

def visible(tortue):
    return tortue.xcor() >= -taille and \
           tortue.xcor() <=  taille and \
           tortue.ycor() >= -taille and \
           tortue.ycor() <=  taille

def deplacer(t, x, y):
    t.goto(x-300+x_pos_axe, y-300+y_pos_axe)

# Grille
t = turtle.Turtle()
t.speed(0)
t.hideturtle()
t.width(1)
t.color('yellow')

def create_line(x0, y0, x1, y1):
    t.penup()
    t.goto(x0, y0)
    t.pendown()
    t.goto(x1, y1)

i = -taille
while i <= taille:
    create_line(-taille, i, taille, i)
    create_line(i, -taille, i, taille)
    i = i + 10

# Repere
t.color('black')
create_line(-taille, -taille + y_pos_axe, taille, -taille + y_pos_axe)
create_line(-taille + x_pos_axe, -taille, -taille + x_pos_axe, taille)
t.penup()
deplacer(t, 380, 5)
t.write("x")
deplacer(t, 5, 380)
t.write("y")

# Mise en place des tortues
tortue = turtle.Turtle()
tortue.shape("turtle")
tortue.penup()
tortue.speed(0)

point_rouge = turtle.Turtle()
point_rouge.pencolor("red")
point_rouge.penup()
point_rouge.speed(0)

deplacer_depart(10, 20)
point_rouge.write("  A")

# les 4 cibles (merci patron)
coords_cibles = (250,  390), \
                (250,  200), \
                (250,   20), \
                (250, -100)

drawer = turtle.Turtle()
for i, coords in enumerate(coords_cibles):
    x, y = coords
    drawer.penup()
    drawer.speed(0)
    deplacer(drawer, x+15, y-10)
    drawer.write("BCDE"[i])
    deplacer(drawer, x, y)
    drawer.dot(10, "lightgreen")
    drawer.dot(1, "black")
deplacer(drawer, -1000, 0)

def marche(x, y):
    tortue.forward(x)
    tortue.left(90)
    tortue.forward(y)
    tortue.right(90)

# Vitesse de 1 à 50, 0 pour le plus vite possible
tortue.speed(8)
point_rouge.speed(0)
tortue.pendown()
point_rouge.pendown()

turtle.done()

point_rouge.goto(tortue.position())

# Valeurs à changer
Δx = 10
Δy = 10

# Déplacement
while visible(tortue):
    marche(Δx, Δy)

Exercice

Pour chacune des quatre cibles :

  • déterminer le coefficient directeur de la droite passant par la cible et le point rouge ;
  • faire apparaître sur l'écran une première ligne rouge - puis une deuxième - reliant le point rouge à la cible en exécutant le programme avec des valeurs bien choisies des variables Δx et Δy (noter les valeurs qui fonctionnent sur le cahier de recherche).
