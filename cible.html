Atteindre la cible

Au départ, la tortue se trouve sur le point rouge et est orientée vers la droite.

Dans le programme ci-dessous, à chaque passage dans la boucle, la tortue effectue :

    • un certain déplacement horizontal avec tortue.forward(...),
    • tourne à gauche (elle s'oriente donc vers le haut) avec tortue.left(90),
    • effectue un certain déplacement vertical avec tortue.forward(...),
    • puis tourne à droite (elle s'oriente donc vers la droite) avec tortue.right(90).

Le trajet de la tortue est dessiné en noir jusqu'à atteindre le bord de la fenêtre.

Ensuite une ligne reliant le point de départ aux points où la tortue tourne à droite est dessinée en rouge.

Exercice

  1. Dans le programme ci-dessous, modifiez les valeurs des déplacements vers la droite et vers le haut (par exemple 8 et 12 au lieu de 10 et 10) de manière à atteindre la cible verte avec la ligne rouge.
  2. Une fois que vous avez réussi, notez les valeurs qui conviennent sur votre cahier de recherche.
  3. Recommencez et essayez de trouver d'autres valeurs qui conviennent.

import turtle
turtle.set_defaults(
    canvwidth = 600,
    canvheight = 600,
    turtle_canvas_wrapper = document['turtle-div']
   # , turtle_canvas_id = 'turtle-canvas' # default, could be overriden
  )
# Espace de jeu
taille = 300

def visible(tortue):
    return tortue.xcor() >= -taille and \
           tortue.xcor() <=  taille and \
           tortue.ycor() >= -taille and \
           tortue.ycor() <=  taille

def lancer_animation():
    turtle.done()

# Grille
t = turtle.Turtle()
t.speed(0)
t.hideturtle()
t.width(1)
t.color('yellow')

def create_line(x0, y0, x1, y1):
    t.penup()
    t.goto(x0, y0)
    t.pendown()
    t.goto(x1, y1)

i = -taille
while i <= taille:
    create_line(-taille, i, taille, i)
    create_line(i, -taille, i, taille)
    i = i + 10

# Mise en place des tortues
tortue_depart = turtle.Turtle()
tortue_depart.shapesize(5)
tortue_depart.shape("circle")
tortue_depart.color("red")
tortue_depart.penup()
tortue_depart.speed(0)
tortue_depart.goto(-290, -280)

tortue_cible = turtle.Turtle()
tortue_cible.shapesize(5)
tortue_cible.shape("circle")
tortue_cible.color("green")
tortue_cible.penup()
tortue_cible.speed(0)
tortue_cible.goto(0, 90)

tortue = turtle.Turtle()
tortue.shape("turtle")
tortue.penup()
tortue.speed(0)
tortue.goto(-290, -280)
tortue.pendown()

point_rouge = turtle.Turtle()
point_rouge.shapesize(0.01)
point_rouge.pencolor("red")
point_rouge.penup()
point_rouge.speed(0)
point_rouge.goto(-290, -280)
point_rouge.pendown()

# Vitesse de 1 à 50, 0 pour le plus vite possible
tortue.speed(8)
point_rouge.speed(0)

turtle.done()

# Valeurs à changer
deplacement_horizontal = 10
deplacement_vertical = 10

# Déplacement
while visible(tortue):
    tortue.forward(deplacement_horizontal)
    tortue.left(90)
    tortue.forward(deplacement_vertical)
    tortue.right(90)

point_rouge.goto(tortue.position())

lancer_animation()
